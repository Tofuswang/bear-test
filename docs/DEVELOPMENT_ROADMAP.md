# Bearless MVP 開發路線圖

## 專案概述
Bearless MVP 是一個模仿 iOS App 體驗的 Web App，專注於中華民國身分證的數位身份驗證。透過 MyData 連接，提供國籍、年齡、戶籍地三項核心驗證功能。

- [ ] 設定開發環境 (ESLint, Prettier, 熱重載)
- [ ] 建立 Git 版本控制流程

## 風險評估與應對

### 技術風險
- **MyData API 整合複雜度**：準備 Mock 資料和模擬環境
- **iOS Safari 兼容性問題**：提前測試和準備 Polyfill
- **PWA 功能限制**：研究替代方案和降級策略

### 時程風險
- **功能範圍過大**：優先實作核心功能，次要功能可延後
- **技術難題耗時**：預留緩衝時間，準備簡化方案

### 品質風險
- **用戶體驗不夠原生**：持續測試和優化，參考真實 iOS App
- **效能問題**：定期效能測試，及早發現和解決
