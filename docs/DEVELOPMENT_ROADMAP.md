# Bearless MVP 開發路線圖 (前端展示版)

## 專案概述
Bearless MVP 是一個模仿 iOS App 體驗的 Web App 前端展示，專注於中華民國身分證的數位身份驗證 UI/UX。**使用假資料和模擬流程**，完美呈現 iOS 原生 App 的視覺體驗和互動感受。

## 設計原則指導

本開發路線圖遵循 `DESIGN_PRINCIPLES.md` 中定義的核心原則：
- **個人身分主權**：用戶完全掌控自己的身分資料
- **選擇性揭露**：只揭露必要的最小資訊
- **去中心化儲存**：避免集中式資料庫監控風險
- **用戶權利保護**：不強制使用，隨時可退出
